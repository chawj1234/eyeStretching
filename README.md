# 👁️ 눈 스트레칭 앱 (Eye Stretching App)

iOS 눈 추적 기능을 활용한 혁신적인 눈 건강 관리 앱입니다. 움직이는 포인트를 따라가며 자연스럽게 눈 근육을 스트레칭할 수 있습니다.

## 🎮 사용법

### 1단계: 시작하기
1. 앱을 실행하고 "스트레칭 시작" 버튼을 탭합니다
2. 3초 카운트다운이 시작됩니다
3. "준비하세요!" 메시지가 나타나면 화면을 주시합니다

### 2단계: 포인트 따라가기
1. 파란색 점이 나타나면 눈으로 따라가세요
2. 점선으로 표시된 경로를 참고하세요
3. 자연스럽게 눈을 움직이며 포인트를 추적합니다

### 3단계: 패턴 완주
1. **원형 패턴** (30초): 시계방향으로 원을 그리며 움직임
2. **8자 패턴** (30초): 무한대(∞) 모양으로 움직임
3. 총 4사이클(2분)을 완주하면 운동 완료

### 4단계: 완료 및 통계
1. "수고하셨습니다!" 메시지와 함께 완료
2. 건강한 눈을 위한 팁 확인
3. 통계가 자동으로 업데이트됩니다
